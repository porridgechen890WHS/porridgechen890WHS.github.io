git tag

软件发布一个版本后，一般会打个tag，标记这个版本，方便回溯。

git tag列出已有的tag
git tag -l "v1.*"使用通配符过滤所有tag
git show v1.0查看v1.0的详细信息
git tag v1.0新建tag
git tag -a v1.0 -m "版本1.0"新建tag并添加备注
git tag -a v1.0 abcdef -m "版本1.0"给指定的某次提交新建tag并添加备注
git push origin --tags推送本地所有tag
git push origin v1.0推送单个tag
